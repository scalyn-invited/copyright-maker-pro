Dynamic copyright shortcode with live preview, styling controls, output formatting, and Legal Link Support.

== Description ==

Copyright Maker Pro provides a configurable copyright shortcode:

[cmp_copyright]

Example output:

&copy; 2026 Your Company | All Rights Reserved | Privacy Policy | Terms & Conditions

Features:

* Auto current year using the WordPress site timezone
* Optional Start Year field for year ranges
* Editable company name
* Optional company-name homepage link
* Editable custom text / rights text
* Symbol Options: copyright, registered, trademark, custom
* Separator Options: pipe, dash, em dash, bullet, slash, custom
* Styling Controls: font family, responsive font sizes, font weight, text color, link color, hover color, letter spacing, and text transform
* Legal Link Support with add/delete link rows
* Per-link Open in New Tab option
* Legal Link Divider options
* Responsive layout controls for Desktop, Tablet, and Mobile
* Live Preview that updates while editing
* Copy shortcode button
* Admin hero banner with one-click shortcode copy
* Dismissible Quick Tips panel (remembered per user)

== Installation ==

1. Upload the plugin ZIP through Plugins > Add New > Upload Plugin.
2. Activate Copyright Maker Pro.
3. Go to Copyright Maker in the admin sidebar.
4. Configure settings.
5. Use [cmp_copyright] where you want the copyright to appear.

== Frequently Asked Questions ==

= What shortcode do I use? =

Use [cmp_copyright]. The compatibility shortcode [copyright_maker] is also supported.

= Can I add Privacy Policy and Terms links? =

Yes. Enable Legal Link Support, add each link name and URL, then save.

= Can links open in a new tab? =

Yes. Each legal link row has its own New Tab checkbox.

= Can I control hover color? =

Yes. Open Styling Controls, use Hover Color, then save. The WordPress color picker is enabled for Text Color, Link Color, and Hover Color.
